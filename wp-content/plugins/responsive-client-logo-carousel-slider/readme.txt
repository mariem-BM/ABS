== Description ==

Responsive Client Logo Carousel Slider Is a nice wordpress plugin which can be used to showcase your client logo in a professional way. 


	
= Features =
* You can create unlimited carousel
* Any types of image are supported
* Shortcode driven, So that you can place anywhere of wordpress such as post, page, widgets and template
* Control the Direction with mouse pointer
* Control the speed, behavior, width, height, border , background and more....
* User friendly interface
* Works on all major browsers - IE7, IE8, IE9, Safari, Firefox, Chrome
* LIte weight and super fast

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `plugin-directory` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode in page, post or in widgets.
4. If you want player in your theme php, Place `<?php echo do_shortcode('YOUR_SHORTCODE'); ?>` in your templates
